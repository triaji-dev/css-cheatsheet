CSS Declarations

There are many CSS declarations available to use. However, not all of them are necessary for day-to-day work. In fact, you can create attractive websites by understanding just some of these CSS declarations.

Here are the CSS declarations that you will actually use.

Color and Background

1. color

Examples 
color: #000000; /* black */
color: rgb(0, 0, 0); /* black */
color: rgba(0, 0, 0, 1); /* black with full opacity */

2. background-color

Examples 
background-color: #ffffff; /* white */
background-color: rgb(255, 255, 255);
background-color: rgba(255, 255, 255, 0.5);

3. background-image

Examples 
background-image: url('image.jpg');
background-image: linear-gradient(to right, red, blue);
background-image: radial-gradient(circle, yellow, green);

4. background-repeat

Examples 
background-repeat: repeat; /* default */
background-repeat: no-repeat;
background-repeat: repeat-x;
background-repeat: repeat-y;

5. background-position

Examples 
background-position: center;
background-position: top right;
background-position: 50% 50%;
background-position: 10px 20px;

6. background-size

Examples 
background-size: cover;
background-size: contain;
background-size: 100px 100px;
background-size: 50% auto;

7. background-attachment

Examples 
background-attachment: scroll; /* default */
background-attachment: fixed;
background-attachment: local;

align-content: stretch; /* default */
align-content: flex-start;

flex-basis: auto; /* default */

/*
shorthand for:

*/

11. order

order: 0; /* default */
order: 1;

column-gap: 10px;
column-gap: 1rem;
column-gap: 20px;

row-gap: 10px;

row-gap: 20px;

grid-column: 1 / 3;

grid-row: 1 / 2;

CSS Variables

1. Defining Variables

:root {
  --main-color: #06c;
  --padding: 10px;
}

2. Using Variables

element {
  color: var(--main-color);
  padding: var(--padding);
}

Transforms & Transitions

1. transform (translate)

transform: translateX(50px);
transform: translateY(20px);
transform: translate(50px, 20px);

2. transform (rotate)

transform: rotate(45deg);
transform: rotate(-90deg);

3. transform (scale)

transform: scale(1.5);
transform: scaleX(2);
transform: scaleY(0.5);
transform: scale(2, 0.5);

4. transform (skew)

transform: skewX(20deg);
transform: skewY(10deg);
transform: skew(20deg, 10deg);

5. transform (multiple)

transform: translate(50px, 50px) rotate(45deg) scale(1.2);

6. transition

transition: all 0.3s ease;
transition: transform 0.5s ease-in-out;
transition: opacity 0.3s, transform 0.3s;

7. transition-property

transition-property: opacity;
transition-property: all;
transition-property: transform, opacity;

8. transition-duration

transition-duration: 0.3s;
transition-duration: 300ms;

Animations

1. animation

animation: mymove 5s infinite;
animation: slide 1s ease-in-out;

2. animation-name

animation-name: mymove;

3. animation-duration

animation-duration: 5s;
animation-duration: 500ms;

4. @keyframes

@keyframes mymove {
  from {top: 0px;}
  to {top: 200px;}
}

@keyframes slide {
  0% { transform: translateX(0); }
  50% { transform: translateX(100px); }
  100% { transform: translateX(0); }
}
